**Retro Achievements Missables Guide** <br> 🔶 ***_Zork III_*** <sub><sup> (_Apple II_) </sup></sub>

This guide covers missable and optional achievements for [Zork III on Retroachievements](https://retroachievements.org/game/18585). For progression achievements, you can refer to [Zork III Invisiclues](https://zedlopez.github.io/invisiclue/zorkiii.html). For all other achievements, read on!

Since Zork III is a puzzle game, this guide uses a system similar to Invisiclues where you can get hints rather than an outright solution for any given achivement. Click on the question you have for the achievement, and it should display the next hint. If you're still stuck you can keep clicking to expand the next hints until you'll eventually see the solution, which will appear in bold.

Set and guide created by [stoneskipper42](https://retroachievements.org/user/stoneskipper42)<br>
Badges created by [DrPac](https://retroachievements.org/user/DrPac)<br>
Information in this guide courtesy of:<br>
[Zork III Source Code Collection](https://github.com/historicalsource/zork3)<br>
[Zork Wiki](https://zork.fandom.com/wiki/)

This guide mirrors the one available on the achievement set developer's [Github](https://github.com/stoneskipper42/RAScripts/tree/main/18585%20(Zork%20III))

```
Live Die Repeat [1]
Restore a saved game immediately after dying. Don't forget to save early and often!
```
<br>

<details> 
<summary>How do I save?</summary>
<!-- have to be followed by an empty line! -->
  <details>
  <summary><i>First thing it says to do is insert a save diskette, so insert a formatted save disk in drive 1 or 2.</i></summary>
  <!-- have to be followed by an empty line! -->    
    <details>
    <summary>I don't have one. How do I make one?</summary>
    <!-- have to be followed by an empty line! -->
      <details>
      <summary><i>AppleWin's documentation says you can create a new disk image that acts as a blank, unformatted by telling AppleWin to use an image file which doesn't already exist. But you'd need to format it before it can be used.</i></summary>
      <!-- have to be followed by an empty line! -->
        <details>
        <summary>How do I format the disk?</summary>
        <!-- have to be followed by an empty line! -->
          <details>
          <summary><i>AppleWin's documentation says you can use the DOS 3.3 disk image to write and initialize a program to format the disk. Assuming you have DOS 3.3.</i></summary>
            <details>
            <summary><i>I wonder if there's any other formatted Apple II disks you could use?</i></summary>
            <!-- have to be followed by an empty line! -->
              <b>Make a copy of your Zork III disk and rename it.</b>
            </details>
          </details>
        </details>
      </details>
    </details>
  </details>
</details>


<details> 
<summary>Oops, I saved over my Zork III disk and now it won't boot. How do I fix it?</summary>
<!-- have to be followed by an empty line! -->
  <details>
  <summary><i>AppleWin doesn't actually overwrite your dsk images, it creates its files for "modified" disks and adds them to a SAVE directory wherever you have AppleWin installed. As long as you have a Zork III disk with the same name and AppleWin has a "modified" disk with the same name, it'll load your corrupted disk.</i></summary>
  <!-- have to be followed by an empty line! -->    
    <details>
    <summary>Oooh, that's so interesting ... but really, how do I fix it?</summary>
    <!-- have to be followed by an empty line! -->
      <b>Rename your Zork III disk or delete the modified Zork III image in AppleWin's SAVE directory.</b>
    </details>
  </details>
</details>

<details> 
<summary>Okay, I have a save diskette but every time I try to save it says "Failed". What am I doing wrong?</summary>

  <!-- have to be followed by an empty line! -->    
    <details>
    <summary><i>Errr, the manual says to follow the instructions for saving and restoring on the Reference Card. Where's the reference card?</i></summary>
    <!-- have to be followed by an empty line! -->
      <details>
      <summary><i>Finding a readable copy of the INTERLOGIC Reference Card sure seems difficult, doesn't it?</i></summary>
      <!-- have to be followed by an empty line! -->
        <b>You can use any position (0-7), but you HAVE to use slot 6. Make sure you've loaded a formatted save disk into drive 2.</b>
      </details>
    </details>
  </details>
</details>


<details> 
<summary>Hey, what's the deal? I died and restored, but the acheivement didn't pop!</summary>
<!-- have to be followed by an empty line! -->
  <details>
  <summary><i>Make sure you restored immediately after dying, without making any other moves. Trying to go directions you can't, looking around, or checking your inventory still count as moves.</i></summary>
  <!-- have to be followed by an empty line! -->

    <summary>Okay, I entered "RESTORE" immediately after dying and before any other commands, but it still didn't pop!</summary>
    <!-- have to be followed by an empty line! -->

      <!-- have to be followed by an empty line! -->
        <details>
        <summary>I did that too! It's still not working!</summary>
        <!-- have to be followed by an empty line! -->
          <b>Use F11 to create save states of your game right before you restore and right after, then <a href="https://retroachievements.org/achievement/612753/report-issue">open a ticket</a> and I'll take a look.</b>
